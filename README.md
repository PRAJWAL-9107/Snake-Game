# Snake Game

## Description

This is a classic Snake Game developed using **HTML, CSS, and JavaScript**. The player controls a snake that moves around the game board, collecting food to grow longer and increase the score. The game ends if the snake collides with the wall or its own body.

## Features

* Interactive gameplay with keyboard controls.
* Random food generation.
* Snake grows after consuming food.
* Live score tracking.
* Automatic game over detection.
* Increasing game speed for added difficulty.
* Clean and simple user interface.
* Lightweight and easy to run in any modern web browser.

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## Game Controls

* **⬆ Up Arrow** – Move Up
* **⬇ Down Arrow** – Move Down
* **⬅ Left Arrow** – Move Left
* **➡ Right Arrow** – Move Right

## Project Structure

* `index.html` – Contains the game layout.
* `style.css` – Handles the game's design and styling.
* `script.js` – Implements the game logic, movement, scoring, and collision detection.
